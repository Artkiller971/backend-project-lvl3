A tool for downloading web pages with assets 

## Installation

```
git clone git@github.com:Artkiller971/backend-project-lvl3.git page-loader
cd page-loader
npm link
```

## Usage

General usage `page-loader <url>`

Use -o flag to choose output directory (default is current): `page-loader -o <directory> <url>`
